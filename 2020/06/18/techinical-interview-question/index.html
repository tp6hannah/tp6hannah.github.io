What is overloading?

overloading means a class can have more than one method having the same name, but with different number or different type of arguments.

What is overriding?

when a subclass has a exactly same method (same name, same number of argument and argumet are all same type same order) as parent class, then the method in the subclass is said to override the method in the parent class.
