What is Angular

Angular is a javascript framework and development platform for creating single-page apps.
The javascript code in Angular mainly use typescript, and typescript is a programming lanaguage which looks like javascript but has other functoin like following

  • make javascript variable use strong type
  • compile the code to javascript
  • make javascript has the behavior like other object oriented programming language -> class, interface

Requirement & Prerequisites

Machine

  • nodejs, which include npm
    you can install nodejs by brew

Human

  • Basic knowledge of HTML, CSS and Javascript

Get Angular CLI from npm to start a angular project

and the result will similiar like this

start a new project

folder structure explaination

open the project in VScode or other IDE

you can see many resources which were created by angular cli

first of all in src folder, which are the main place we code our application

  • src/app -> code the html, css, js in here
  • src/assets -> put the image or other material which the applicaion will use
  • src/environments -> config for dev and production if needed
  • src/tsconfig.json -> help our typescript code convert to javascript
  • src/test.ts -> starting point for unit test

Exploring Angular CLI Commands

  • ng lint helps us for checking the coding standard, and the standard is defined at tslint.json

  • ng build will generate the a folder called dist, in it are the compiled source file to let you put on web server

  • ng test run the unit test for angular app
    you can write the test case in the file which has the extention like *.spec.ts

  • ng e2e run the end-to-end test for your angular application
    you can write the e2e test in e2e/ folder

explore other angular cli command, you can ng help

and last but not least, try ng serve and reuqest localhost:4200 in you broswer, then you can see what default angular application looks like!!
