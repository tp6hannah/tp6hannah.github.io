Posted on 2020-05-24 In Java

Objective

Describe common usages related to Map, HashMap class in Java.

Basic

In Java, Map is an interface, which is in java.util package

  • An object that maps keys to values.
  • A map cannot contain duplicate keys, each key can map to at most one value.
  • A member of the Java Collections Framework.

can implement Map interface by many class, such as HashMap class, which is the most common one, or HashTable, TreeMap etc.
